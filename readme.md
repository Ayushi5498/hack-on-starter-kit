# Hackathon Starter Kit

Next.js + Supabase + Gemini — ready to extend for any hackathon problem.

## Quick Start
1. Clone repo
2. Copy .env.example → .env.local and fill values
3. npm install
4. npm run dev

## Environment Variables
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
GEMINI_API_KEY=

## What's Included
- Auth (login/signup/logout)
- Database CRUD
- File storage
- AI (Gemini)
- Dashboard layout
- Charts
- Chat UI
- Dark mode

## Repository Files
- README.md → Setup and overview
- PROJECT_CONTEXT.md → KIRO architecture context (read this first)
- TOPIC_SELECTION_GUIDE.md → How to pick the best problem on hackathon day
- .env.example → Environment variable template

## KIRO Usage
1. Open this repository in KIRO
2. Read PROJECT_CONTEXT.md
3. Read TOPIC_SELECTION_GUIDE.md
4. Paste the problem statements using the prompt template inside TOPIC_SELECTION_GUIDE.md
